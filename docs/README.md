## 简介
睡眠监测平台，架构、业务、技术要点全方位解析。sleep-monitoring-platform 项目是一套健康监测平台系统，使用现阶段主流技术实现。  
涵盖了SpringBoot 2.4.0、MyBatis 3.4.6、Elasticsearch 7.6.2、Redis 5.0、Mysql5.7等技术，采用Docker容器化部署。
